## 🎮 Overview

**Moon's Shadow** is a single-player open-world exploration game built with Unity. The player navigates through dense forest environments, solves environmental puzzles, and progresses toward a mysterious temple as the narrative endpoint.

Inspired by the exploration and environmental storytelling of *The Legend of Zelda: Breath of the Wild*, the project focuses on atmosphere, spatial design, and player-driven discovery.

## 🚀 Getting Started

### Prerequisites

- Unity 6 (LTS recommended)
- Git

### Installation

```bash
git clone https://github.com/dev-solinor/Moon-s-Shadow.git
```

Open the project in Unity Hub, then load the main scene from `Assets/_Project/Scenes/`.
